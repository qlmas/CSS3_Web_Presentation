Osnovni Selektori

CSS selektori omogućavaju stilizovanje određenih elemenata na web stranici. Postoje razne vrste selektora koji omogućavaju precizno ciljanje HTML elemenata.
Evo nekih od najvažnijih:

Element selektori

Odabir elemenata po imenu taga (npr. p odabira sve paragrafe).

p { color: blue; }

To bi izgledalo ovako!

Class selektori

Odabir elemenata sa određenom klasom (npr. .classname).

.classname { font-size: 20px; }

Ovaj tekst je veličine dvadeset piksela.

ID selektori

Odabir elemenata sa određenim ID-om (npr. #idname).

#idname { background-color: yellow; }

Ovaj tekst ima žutu boju pozadine.

Kompleksni Selektori

Kompleksni CSS3 selektori omogućavaju ciljanje elemenata na osnovu njihove pozicije, odnosa s drugim elementima ili specifičnih atributa.

Descendant selektori

Odabir elemenata koji su potomci određenog elementa (npr. div p odabira sve paragrafe unutar divova).

div p { margin: 10px; }
Child selektori

Odabir direktnih potomaka određenog elementa (npr. div > p odabira sve paragrafe koji su direktna 'djeca' divova).

div > p { padding: 5px; }
Attribute selektori

Odabir elemenata sa određenim atributima (npr. input[type="text"] odabira sve inpute tipa tekst).

input[type="text"] { border: 1px solid #000; }

Pseudo-klase i Pseudo-elementi

Pseudo elementi se koriste za stilizovanje određenih djelova elemenata, kao što su prvo slovo, prvi red ili sadržaj umetnut prije ili poslije elementa.

Pseudo-klase

Odabir elemenata u određenom stanju (npr. a:hover odabira linkove kada se prelazi mišem preko njih).

a:hover { color: red; }

Kada preko ovog pređete postat će crveno!

Pseudo-elementi

Odabir i stilizovanje djelova elemenata (npr. p::first-line odabira prvu liniju paragrafa).

p::first-line { font-weight: bold; }

Ovo je prva linija
ovog paragrafa
i samo će ona biti boldirana.

Napredni Selektori u CSS3

Attribute selektori

Prošireni su u CSS3 da uključuju više mogućnosti kao što su [attribute^="value"], [attribute$="value"], i [attribute*="value"].

a[href^="https"] { color: green; }
:nth-child() i :nth-of-type()

Selektovanje elemenata na osnovu njihovog položaja u grupi.

li:nth-child(odd) { background: #eee; }
:not()

Selektovanje elemenata koji ne odgovaraju određenom selektoru.

:not(p) { color: grey; }
