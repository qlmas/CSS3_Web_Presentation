Animacije i Tranzicije

CSS3 uvodi moćne alate za animacije i tranzicije koje omogućavaju stvaranje vizuelno atraktivnih efekata na web stranicama. Ove funkcije poboljšavaju korisničko iskustvo pružajući elegantne i interaktivne animacije.

Tranzicije

Tranzicije omogućavaju elegantne promjene stilova elemenata tokom određenog vremena. Koriste se za kreiranje jednostavnih animacija kada se atributi CSS-a mijenjaju.


.box {
    width: 100px;
    height: 100px;
    background-color: #35A8DB;
    transition: background-color 0.5s ease;
}

.box:hover {
    background-color: #1572B6;
}

Primjer :hover efekta

Animacije

CSS animacije omogućavaju definiranje kompleksnih sekvenci ključnih frejmova koji mijenjaju stilove elemenata tokom vremena.


@keyframes example {
    0% { background-color: #1572B6; left: 0px; }
    12.5% { background-color: #1F80BC; left: 200px; }
    25% { background-color: #298EC2; left: 400px; }
    37.5% { background-color: #339CCA; left: 200px; }
    50% { background-color: #3DAAD0; left: 0px; }
    62.5% { background-color: #339CCA; left: -200px;}
    75% { background-color: #298EC2; left: -400px;}
    87.5% { background-color: #1F80BC; left: -200px;}
    100% { background-color: #1572B6; left: 0px; }
                }

.animation-example .animated-box {
    position: relative;
    width: 100px;
    height: 100px;
    background-color: blue;
    animation: example 8s infinite;
                }

Transformacije

Transformacije omogućavaju translaciju, skaliranje, rotaciju i kose elemente. One su često korišćene zajedno sa tranzicijama i animacijama za stvaranje kompleksnih efekata.


.transform-box {
    width: 100px;
    height: 100px;
    background-color: #35A8DB;
    transform: rotate(45deg);
    transition: transform 0.5s ease;
}

.transform-box:hover {
    transform: rotate(0deg);
}
