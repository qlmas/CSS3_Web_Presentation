Animacije i Tranzicije

CSS3 uvodi moćne alate za animacije i tranzicije koje omogućavaju stvaranje vizuelno atraktivnih efekata na web stranicama. Ove funkcije poboljšavaju korisničko iskustvo pružajući elegantne i interaktivne animacije.

Tranzicije

Tranzicije omogućavaju elegantne promjene stilova elemenata tokom određenog vremena. Koriste se za kreiranje jednostavnih animacija kada se atributi CSS-a menjaju.


.box {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: background-color 0.5s ease;
}

.box:hover {
    background-color: red;
}

Primjer :hover efekta

Animacije

CSS animacije omogućavaju definiranje kompleksnih sekvenci ključnih frejmova koji mijenjaju stilove elemenata tokom vremena.


@keyframes example {
    0% { background-color: blue; left: 0px; top: 0px; }
    25% { background-color: green; left: 200px; top: 0px; }
    50% { background-color: yellow; left: 200px; top: 200px; }
    75% { background-color: red; left: 0px; top: 200px; }
    100% { background-color: blue; left: 0px; top: 0px; }
}

.animated-box {
    position: relative;
    width: 100px;
    height: 100px;
    background-color: blue;
    animation: example 4s infinite;
}

Primjer animacije

Transformacije

Transformacije omogućavaju translaciju, skaliranje, rotaciju i kose elemente. One su često korišćene zajedno sa tranzicijama i animacijama za stvaranje kompleksnih efekata.


.transform-box {
    width: 100px;
    height: 100px;
    background-color: purple;
    transform: rotate(45deg);
    transition: transform 0.5s ease;
}

.transform-box:hover {
    transform: rotate(0deg);
}
