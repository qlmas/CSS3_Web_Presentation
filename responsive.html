Responsive Design

Responsive design omogućava da web stranica izgleda dobro na svim uređajima (desktop računarima, tabletima i mobilnim telefonima) koristeći CSS3 media queries i fleksibilne mreže (grids). Cilj je da stranica bude lako čitljiva i navigabilna bez obzira na uređaj.

Media Queries

Media queries omogućavaju promjenu stilova na osnovu karakteristika uređaja kao što su širina, visina, rezolucija i orijentacija ekrana.


/* Primjer za mobilne uređaje */
@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
    .container {
        flex-direction: column;
    }
}

/* Primjer za tablete */
@media (min-width: 601px) and (max-width: 1024px) {
    body {
        background-color: lightgreen;
    }
    .container {
        flex-direction: row;
    }
}

/* Primjer za desktop uređaje */
@media (min-width: 1025px) {
    body {
        background-color: lightcoral;
    }
    .container {
        flex-direction: row;
    }
}

Fleksibilne mreže (Flexbox i Grid)

Flexbox i CSS Grid Layout omogućavaju kreiranje fleksibilnih i responzivnih rasporeda koji se lako prilagođavaju različitim veličinama ekrana.

Flexbox Primjer


.container {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
}
.item {
    flex: 1 1 200px; /* Rasteže se i zauzima minimalno 200px */
    margin: 10px;
}

Grid Primjer


.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 10px;
}
.grid-item {
    background-color: #ccc;
    padding: 20px;
}

Responzivne slike

Responzivne slike omogućavaju promjenu veličine slika u zavisnosti od veličine ekrana.


img {
    max-width: 100%;
    height: auto;
}

Viewport Meta Tag

Viewport meta tag omogućava kontrolu nad izgledom stranice na mobilnim uređajima. Postavlja se u <head> sekciji HTML dokumenta.


<meta name="viewport" content="width=device-width, initial-scale=1.0">
