Layout Tehnike

CSS3 donosi mnoge nove tehnike za layout koje omogućavaju kreiranje kompleksnih i responzivnih web dizajna. Ove tehnike uključuju Flexbox, Grid, i druge metode za kontrolu rasporeda elemenata na stranici.

Flexbox

Flexbox (Flexible Box) je layout model koji omogućava efikasno raspoređivanje prostora među stavkama unutar kontejnera, čak i kada su njihova veličina nepoznata i/ili dinamična.

 
.container {
    display: flex;
    flex-direction: row; /* ili column */
    justify-content: center; /* Poravnanje po glavnoj osi */
    align-items: center; /* Poravnanje po poprečnoj osi */
}

Grid Layout

CSS Grid Layout je dvodimenzionalni layout sistem koji omogućava kreiranje kompleksnih layouta sa redovima i kolonama.

 
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Tri kolone jednake širine */
    grid-template-rows: auto;
    gap: 10px; /* Razmak između grid stavki */
}
.item {
    grid-column: span 2; /* Stavka koja se prostire preko dve kolone */
}

Float i Clear

Float svojstvo se koristi za raspoređivanje elemenata levo ili desno unutar kontejnera, dok clear svojstvo sprečava elemente da se pojave pored floated elemenata.

 
img {
    float: left;
    margin: 10px;
}
p.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

Ovo je primer teksta pored slike koja koristi float.

Positioning

CSS positioning omogućava precizno pozicioniranje elemenata na web stranici.

  • Static: Default pozicioniranje.
  • Relative: Pozicioniranje relativno prema normalnoj poziciji.
  • Absolute: Pozicioniranje relativno prema najbližem pozicioniranom pretku.
  • Fixed: Pozicioniranje relativno prema prozoru preglednika. Klikom ovdje aktivirate fixed element.
  • Sticky: Pozicioniranje bazirano na scroll poziciji.
